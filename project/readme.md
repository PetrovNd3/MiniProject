# Ticket Booking 
A ticket booking application.

## FEATURES 
- Two kinds of users - Normal user and Admin
#### Admins: 
- Add new movies
- Delete movies
- Delete venues
- View different movies
#### Normal Users:
- Login / Signin.
- Able to view movie details
- Able to book seats.
